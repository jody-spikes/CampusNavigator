# Campus Navigator

CSI 3334 Spring 2026 Group Project — campus navigation and
resource management system in C++.

## Build

    make         # compile
    make run     # compile and run
    make clean   # remove build artifacts

## Requirements

- g++ with C++17 support
- GNU Make
